Display a simple table timetable with different colours for different events.

== Description ==

Use this plugin to display a simple and easy to configure timetable for multiple events. Each event can be color coded so it can be easily distinguished.

== Features ==

* Two header rows
* Fun colors
* Simple customisation


== Installation ==

Either search for 'Timetable' under Plugins on your Wordpress Blog, or manually download it here, or on GitHub and upload it to your Wordpress Blog. Then activate it and configure it.
